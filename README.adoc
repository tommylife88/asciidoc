## Env

### Ruby gems

https://www.ruby-lang.org/

[source,bash]
----
gem install bundler
bundle config set --local path .bundle/.gems
bundle config set --local github.https true
bundle install
----

### Usage

[source,bash]
----
bundle exec rake -T
----
